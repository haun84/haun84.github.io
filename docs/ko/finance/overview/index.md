# 금융시스템

::: tip 금융시스템
**금융시스템**은 **금융시장** 및 **금융기관**과 이들을 형성하고 운영하며 원활하게 기능하도록 하는 **법규**와 **관행**, **지급결제시스템** 등 **금융인프라**를 모두 포괄하는 개념이다.
:::
