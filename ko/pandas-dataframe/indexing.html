Language

데이터프레임 DataFrame 구조

  • 데이터프레임은 공통 인덱스를 가지는 열 시리즈(column series)를 딕셔너리로 묶어놓은 것이다.
  • 2차원 배열 데이터는 모든 원소가 같은 자료형을 가져야 하지만, 데이터프레임은 각 열(column)마다 자료형이 다를 수 있다.
  • 열방향 인덱스와 행방향 인덱스는 각각 columns, index 속성으로 접근한다.

데이터프레임 생성 ​

  1. 우선 하나의 열이 되는 데이터를 리스트나 일차원 배열로 준비한다.
  2. 이 각각의 열에 대한 이름(라벨)을 키로 가지는 딕셔너리를 만든다.
  3. 이 테이터를 DataFrame 클래스 생성자에 넣는다. 동시에 열방향 인덱스는 columns 인수로, 행방향 인덱스는 index 인수로 지정한다.
python
import pandas as pd

# Dictionary
data = {
    'year': [2020, 2021, 2022, 2023],            # int
    'GDP rate': [2.8, 3.1, 3.0, 3.2],            # float
    'GDP': ['1.63M', '1.72M', '1.81M', '1.93M']  # str
}

# 데이터프레임 생성방법1 : 행방향 레이블과 열방향 레이블을 직접 지정
columns = ['year', 'GDP rate', 'GDP']
index = [2020, 2021, 2022, 2023]
df = pd.DataFrame(data, index=index, columns=columns)
print(df)

#       year  GDP rate    GDP
# 2020  2020       2.8  1.63M
# 2021  2021       3.1  1.72M
# 2022  2022       3.0  1.81M
# 2023  2023       3.2  1.93M

# 데이터프레임 생성방법2 : 딕셔너리의 열방향 레이블 사용 및 행방향 레이블 지정
df = pd.DataFrame(data, index=data['year'])  # year 열을 행방향 레이블로 지정
print(df)

#       year  GDP rate    GDP
# 2020  2020       2.8  1.63M
# 2021  2021       3.1  1.72M
# 2022  2022       3.0  1.81M
# 2023  2023       3.2  1.93M

데이터에만 접근하기 ​

python
# 데이터 전체 반환
df.values

#        [2021, 3.1, '1.72M'],
#        [2022, 3.0, '1.81M'],
#        [2023, 3.2, '1.93M']], dtype=object)

열 인덱스 접근하기 ​

python
# 열 인덱스 반환
df.columns

# Index(['year', 'GDP rate', 'GDP'], dtype='object')

행 인덱스 접근하기 ​

python
# 행 인덱스 반환
df.index

# Index([2020, 2021, 2022, 2023], dtype='int64')

데이터프레임 행 수 확인 ​

python
# 데이터프레임 행 수 확인
len(df.index)

열 인덱스와 행 인덱스에 이름 붙이기 ​

python
df.index.name = "Year"
df.columns.name = "Value"
df

# Value year  GDP rate    GDP
# Year

# 2021  2021       3.1  1.72M

# 2023  2023       3.2  1.93M

열 인덱싱 ​

열 인덱싱

  • 데이터프레임은 열 라벨을 키로, 열 시리즈를 값으로 가지는 딕셔너리와 비슷하다.
  • 따라서 데이터프레임은 인덱싱을 할 때도 열 라벨(column label)을 키값으로 생각하여 인덱싱 할 수 있다.
  • 인덱스로 라벨 값을 하나만 넣으면 시리즈(series) 객체가 반환되고 라벨의 배열 또는 리스트를 넣으면 부분적인 데이터프레임이 반환된다.

하나의 열만 시리즈로 반환 ​

python
# 하나의 열만 인덱싱하면 시리즈가 반환된다.
df['year']

# Year
# 2020    2020
# 2021    2021
# 2022    2022
# 2023    2023
# Name: year, dtype: int64

여러 개의 열을 데이터프레임으로 반환 ​

python
# 여러 개의 열을 인덱싱하면 부분적인 데이터프레임이 반환된다.
df[['year', 'GDP rate']]


# Value  year  GDP rate
# Year
# 2020   2020       2.8
# 2021   2021       3.1
# 2022   2022       3.0
# 2023   2023       3.2

하나의 열을 데이터프레임으로 반환 ​

만약 하나의 열만 추출하면서 데이터프레임 자료형을 유지하고 싶다면 원소가 하나인 리스트를 써서 인덱싱하면 된다.

python
# GDP rate 열을 반환하면서 데이터프레임 자료형을 유지
df[['GDP rate']]

# Value  GDP rate
# Year
# 2020        2.8
# 2021        3.1
# 2022        3.0
# 2023        3.2

행 인덱싱 ​

슬라이싱(slicing)을 통한 행 단위 인덱싱 ​

python
# 첫 번째 행까지 반환
df[:1]

# Value  year  GDP rate    GDP
# Year
# 2020   2020       2.8  1.63M
python
# 두 번째 행부터 세 번째 행 전 행까지 반환: 두 번째 행만 반환
df[1:2]

# Value  year  GDP rate    GDP
# Year
# 2021   2021       3.1  1.72M
python
# 두 번째 행부터 네 번째 행 전 행까지 반환: 두 번째, 세번째 행만 반환
df[1:3]

# Value  year  GDP rate    GDP
# Year
# 2021   2021       3.1  1.72M
# 2022   2022       3.0  1.81M

개별 데이터 인덱싱 ​

데이터프레임에서 열 라벨로 시리즈를 인덱싱하면 시리즈가 된다. 이 시리즈를 다시 행 라벨로 인덱싱하면 개별 데이터가 나온다.

python
# 개별 데이터 접근
df['GDP'][2021]

# '1.72M'

열 데이터의 갱신, 추가, 삭제 ​

데이터프레임은 열 시리즈의 딕셔너리로 볼 수 있으므로 열 단위로 데이터를 갱신하거나 추가, 삭제할 수 있다.

열 데이터 추가 ​

python
# 'BaseRate' 라는 이름의 열 추가
df['BaseRate'] = [1.5, 1.6, 1.7, 1.8]
df

# Value  year  GDP rate     GDP  BaseRate
# Year
# 2020   2020       2.8   1.63M       1.5
# 2021   2021       3.1   1.72M       1.6
# 2022   2022       3.0   1.81M       1.7
# 2023   2023       3.2   1.93M       1.8

열 데이터 갱신 ​

python
# 'BaseRate' 열의 데이터 갱신
df['BaseRate'] = [1.8, 1.9, 2.0, 2.1]
df

# Value  year  GDP rate     GDP  BaseRate
# Year
# 2020   2020       2.8   1.63M       1.8
# 2021   2021       3.1   1.72M       1.9
# 2022   2022       3.0   1.81M       2.0
# 2023   2023       3.2   1.93M       2.1

열 데이터 삭제 ​

python
# 'BaseRate' 열 삭제
del df['BaseRate']
df

# Value  year  GDP rate     GDP
# Year
# 2020   2020       2.8   1.63M
# 2021   2021       3.1   1.72M
# 2022   2022       3.0   1.81M
# 2023   2023       3.2   1.93M
