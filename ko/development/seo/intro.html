SEO (검색엔진 최적화) 가이드 ​

Language

  • English Page: 영어 페이지

SEO는 Search Engine Optimization의 약자로 Search Engine, 즉 구글과 같은 검색엔진이 이해하기 쉽도록 홈페이지의 구조를 만들고 매력적이고 유용한 콘텐츠를 작성해 검색 결과 상위에 노출될 수 있도록 하는 작업을 말한다.

검색엔진 알고리즘 ​

검색엔진 (Search Engine)이 선호하는 웹사이트를 만들고 글을 쓰기 위해서는 검색엔진이 어떤 원리로 동작하는지 알아야 한다.

  1. 첫 번째 단계인 크롤링 (Crawling)은 구글봇(웹 크롤러)이 내 웹페이지의 콘텐츠를 복사해서 모든 정보를 수집하고 수집한 정보를 검색엔진으로 가져온다.

  2. 다음 단계로 인덱싱 (Indexing)은 구글봇이 가져온 정보를 주제별로 색인해서 데이터를 보관한다.

  3. 마지막으로 랭킹 (Ranking)은 색인된 콘텐츠를 검색 의도에 맞춰서 순위를 부여한 다음 사용자가 해당 키워드로 검색했을 때 랭킹 순서대로 결과를 제공한다.

이 3단계 프로세스를 통해 구글은 SERP (검색엔진 결과페이지), Search Engine Result Page를 완성한다.

테크니컬 SEO ​

웹사이트의 기술적인 구조는 웹사이트 퍼포먼스에 지대한 영향을 미치기 때문에 최적화 작업이 필수이다. 웹사이트가 아무리 많은 양질의 콘텐츠를 가지고 있다고 해도, 건강하고 탄탄한 기술적 구조를 갖추고 있지 않다면 해당 콘텐츠를 검색엔진은 물론 사용자에게도 효과적으로 전달할 수 없다.

아래는 가장 기본적으로 적용해야 할 테크니컬 SEO 작업이다.

  1. 보안 프로토콜 (HTTPS) 적용하기

  2. 모바일 친화성 갖추기

  3. 페이지 로딩 속도 빠르게 하기

  4. URL을 구조적으로 생성하기

  5. 사이트맵 (sitemap.xml) 생성 및 주기적으로 업데이트 하기

  6. 로봇 배제 표준 파일 (robots.txt) 생성하기

HaunBlog by VitePress

  • HaunBlog는 VitePress라는 빠르고 컨텐츠 중심의 웹사이트를 구축하기 위해 설계된 정적 사이트 생성기 (SSG)를 통해 작성하여 높은 모바일 친화성, 빠른 페이지 로딩 속도, 웹사이트 구조의 검색엔진 최적화를 적용 중이다.

콘텐츠 SEO ​

구글은 웹상에 있는 수조 건의 콘텐츠를 분석하고, 해당 콘텐츠를 유용성과 관련성에 따라 순위를 매긴다. 다르게 말하면, 자신의 콘텐츠가 SEO 적인 가치가 있으려면 (상위 노출을 하려면), 1차적으로 그리고 가장 중요하게도 사용자에게 가치를 전달하는 유용한 콘텐츠여야 한다는 것이다.

아래는 가장 기본적으로 적용해야 할 콘텐츠 SEO 작업이다.

  1. 특정 주제에 대해 풍부한 양의 정보 전달하기

  2. 콘텐츠의 주제를 명확히 설명하는 타이틀 만들기

  3. 메타 태그 활용하기

  4. 제목 태그(H 태그) 사용하기

  5. 이미지에 alt 속성 기재하기
