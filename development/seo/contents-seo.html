Contents SEO Overview ​

  • Content SEO refers to the creation of content that helps improve the ranking of a webpage in search engines.

  • Google analyzes trillions of content on the web and ranks the content according to its usefulness and relevance. In other words, in order for your content to be SEO-like (top exposure), it must be useful content that primarily and most importantly delivers value to users.

  • Below is the most basic content SEO work to be applied.

Delivering a rich amount of information about a particular topic ​

  • Generally, SEO-friendly blog content is recommended to be written in at least 1,500 to 2,000 characters, but content written in more than 3,000 characters can get 3 times more traffic, 4 times more social sharing, and 3.5 times more backlinks, so if you deliver in-depth information on a specific topic research results, you can expect excellent content performance.

  • In addition, it can be recognized as high-quality content as it increases the visitor's stay time and increases the user experience with a rich amount of content.

Create a title that clarifies the topic of the content ​

  • The title in the header of the HTML document should be constructed in detail and concise manner. It should also be a convincing title containing the keywords you want to expose.

  • It is preferable to write the length within 30 to 65 characters in English and within 15 to 32 characters in Korean.

Using Meta Tag ​

  • It is recommended to include the description in the meta tag <meta> of the HTML document because it is also displayed in the search results of the search engine.

  • For description, it is recommended to include a summary of the contents of each page as a whole, and to exclude information that overlaps with the title of the page. Also, a convincing explanation consistent with the search intention is good.

Using the Title Tag (H tag) ​

  • The <H> tag is a title tag. The title is an important part of the text. Search engines also view these <H> tags as very important. If the <H> tag simply represents importance by letter size, the search engine cannot know what content is important.

  • The important point when using the <H> tag is that it should be used properly and structurally. It is not appropriate to write all of the contents with the <H> tag, and search engines are also not preferred.

  • Use the <H1> tag on the page title and you should use only one per page. It is important to write structurally using the H2-H6 tag on the page subtitle.

Write alt properties on an image ​

  • The webpage should be available to the visually impaired as well. Blind people use screen reader to check the website. Check various texts, images, tables, lists, etc. on the webpage and provide them by voice. In SEO, it is necessary to review how friendly they are to screen reader considering the usability of various users.

  • To put it simply, there is an image alternative text (alt attribute). We know what image it is visually, but we need an alt attribute that explains what image it is for users of screen reader. And the Robot of the search engine is also advantageous for SEO because it collects this alt attribute and performs indexing.
