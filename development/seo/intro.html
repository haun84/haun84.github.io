SEO (Search Engine Optimization) Guide ​

SEO stands for Search Engine Optimization, which is a task that creates a homepage structure to make it easier for search engines such as Google to understand and creates attractive and useful content so that they can be exposed to the top of the search results.

Search Engine Algorithm ​

In order to create and write a website that Search Engine prefers, you need to know how the search engine works.

  1. The first step, Crawling, is that a web crawler copies the content of my web page, collects all the information, and brings the collected information to the search engine.

  2. In the next step, Indexing indexes the information brought by Googlebot by subject and stores the data.

  3. Finally, Ranking ranks the indexed content according to the search intention, and then provides results in order of ranking when the user searches with the keyword.

Through this three-step process, Google completes the SERP and Search Engine Results Page.

Technical SEO ​

Optimization is essential because the technical structure of a website has a profound impact on website performance. No matter how much high-quality content a website has, it cannot effectively deliver the content to users as well as search engines, unless it has a healthy and solid technical structure.

Below are the most basic technical SEO tasks to be applied.

  1. Apply Secure Protocol (HTTPS)

  2. Getting Mobile Friendly

  3. Fast Page Loading

  4. To generate a URL structurally

  5. Create a Sitemap (sitemap.xml) and update it periodically

  6. Create Robots Exclusion Standard File (robots.txt)

Contents SEO ​

Google analyzes trillions of content on the web and ranks the content according to its usefulness and relevance. In other words, in order for your content to be SEO-like (top exposure), it must be useful content that primarily and most importantly delivers value to users.

Below are the most basic contents SEO tasks to be applied.

  1. Delivering a rich amount of information about a particular topic

  2. Create a title that clarifies the topic of the content

  3. Using Meta Tag

  4. Using the Title Tag (H tag)

  5. Write alt properties on an image
